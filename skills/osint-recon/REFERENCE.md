# Reference

## Example Calls

```python
sherlock(username="targetuser")
spiderfoot(target="example.com")
sublist3r(domain="example.com")
parsero(target="https://app.example.com")
whois(target="example.com")
```

## Tool Guide

| Tool | Common parameters | Use |
|---|---|---|
| `sherlock` | `username` | username footprint discovery |
| `spiderfoot` | `target` | broad OSINT aggregation |
| `sublist3r` | `domain` | older subdomain collection path |
| `parsero` | `url` | robots.txt analysis |
| `whois` | `domain` | registration and ownership metadata |
