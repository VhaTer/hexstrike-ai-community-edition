**Tips:**
- `nuclei -severity critical` on testphp returned instantly (0 templates match critical-only). Use `low` or `medium` for scanning dead targets.

### Phase 4 Fixes Applied

1. **masscan `rate` type** → Tool registry stores `rate: "1000"` (string), typed wrapper infers `str`. Integration test passed `rate=500` (int) → Pydantic validation error. **Fix:** test now passes `rate="500"`.
2. **dirsearch wordlist path** → config.py pointed to `/usr/share/wordlists/dirsearch/common.txt` (doesn't exist). **Fix:** changed to `/usr/share/wordlists/dirb/common.txt` (4614 entries, exists on this system).
3. **dalfox not installed** → Binary missing. Server auto-suggests `xsser`. **Fix:** replaced with `xsser` in integration tests. Note: xsser itself fails on Python 3.13 (`ModuleNotFoundError: No module named 'cgi'` — removed in 3.13).

## Section 3 — Open Issues

### Resolved
- **masscan Pydantic type error** → rate param now correctly passed as string
- **dirsearch default wordlist** → config path corrected to existing file  
- **Integration test wordlist sizes** → reduced to `small.txt` (959 entries) for ffuf/gobuster/feroxbuster on web target

### Environmental (not code bugs)
- **ffuf timeout (x2)** — both targets unreachable from this network. HTB machine (10.129.95.191) appears powered off; testphp.vulnweb.com port 80 is filtered/blocked. ffuf is correct — it waits for HTTP responses that never come.
- **xsser fails on Python 3.13** — `cgi` module removed from stdlib. This is a system package issue, not a HexStrike bug. Install `dalfox` via Go or use `snap`/`pipx`.
- **masscan permission denied** — requires root/sudo for raw sockets. Expected behavior.
- **testssl.sh not installed** — `testssl` tool binary missing from system. Expected failure, error message is clear.
- **testphp.vulnweb.com unreachable** — all web tools that connect via TCP (whatweb, httpx, gobuster, feroxbuster, nikto, ffuf) fail with connection errors. Only dirsearch and nuclei (which use their own HTTP clients) produced partial results.

### Still Blocked
- **Coverage measurement** — `pytest --cov` crashes with `beartype` circular import on coverage hooks.
- **No HTB VIP access** — limited to free/retired machines that may be powered off.
